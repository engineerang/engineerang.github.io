Setting up Arduino on MacOS for the ESP8266 & ESP32 boards

Date Fri 12 March 2021 Tags arduino / hardware

A quick summary of how I got up and running with the ESP8266 & ESP32 micro-controllers with Arduino on MacOS. Both board types that I have were a bit fiddly so I've included how I got them working below.

Installing Arduino

A good place to start is to install Arduino on MacOS, follow the instructions here.

Setting up for the ESP82660 NodeMCU-Lolin-V3

ESP82660 NodeMCU-Lolin-V Pinout

Board details:

Its important to identify the following features of your board:

  • MCU: NODEMCU 1.0 (ESP - 12E Module)
  • UART: CH340G

Driver installation

Now that we know what the MCU and UART are we can install the appropriate driver. For MacOS I used brew to install the driver as recommended here.

# Install cask
brew tap adrianmihalko/ch340g-ch34g-ch34x-mac-os-x-driver https://github.com/adrianmihalko/ch340g-ch34g-ch34x-mac-os-x-driver
# Install driver
brew cask install wch-ch34x-usb-serial-driver

NOTE: "PySerial and EspTools Directory Error"

To fix this do the follow:

  1. Open ~/Library/Arduino15/packages/esp8266/hardware/esp8266/2.7.4/tools/pyserial/serial/tools/list_ports_osx.py
  2. Comment out lines 29 and 30 and append these lines:
29 | #iokit = ctypes.cdll.LoadLibrary(ctypes.util.find_library('IOKit'))
30 | #cf = ctypes.cdll.LoadLibrary(ctypes.util.find_library('CoreFoundation'))
31 | iokit = ctypes.cdll.LoadLibrary('/System/Library/Frameworks/IOKit.framework/IOKit')
32 | cf = ctypes.cdll.LoadLibrary('/System/Library/Frameworks/CoreFoundation.framework/CoreFoundation')

Arduino Integration

  1. Open Arduino IDE.
  2. Click File > Preference.
  3. Add This URL "http://arduino.esp8266.com/stable/package_esp8266com_index.json" to Additional Board Manager URLs.
  4. Go to Tools > Board > Board Manager.
  5. Wait until the update is complete.
  6. Search esp8266 by ESP8266 Community, then install, wait until the installation is complete.
  7. After it finishes, Go to Tools > Board > NodeMCU 1.0 (ESP-12E Module).

Testing It Out

  1. Select the COM port Tools > Port > "/dev/cu.usbserial-0001".
  2. Click Files > Examples > Basics > blink.
  3. Upload the code to the esp8266 board.
  4. Hit the RST button (look at the pinout diagram).

Setting up for the ESP32 NodeMCU-32S

ESP32 NodeMCU-32S Pinout

Board details:

Its important to identify the following features of your board:

  • MCU: NODEMCU-32S (ESP32)
  • UART: CP2102

Driver installation

Now that we know what the MCU and UART are we can install the appropriate driver. Download and install the driver "CP210x VCP Mac OSX Driver" from here.

Arduino Integration

  1. Open Arduino IDE.
  2. Click File > Preference.
  3. Add This URL "https://dl.espressif.com/dl/package_esp32_index.json" to Additional Board Manager URLs.
  4. Go to Tools > Board > Board Manager.
  5. Wait until the update is complete.
  6. Search esp32 by ESP32mmunity, then install, wait until the installation is complete.
  7. After it finishes, Go to Tools > Board > NodeMCU-32S.

Testing It Out

  1. Select the COM port Tools > Port > "/dev/cu.SLAB_USBtoUART".
  2. Click Files > Examples > Basics > blink.
  3. Upload the code to the esp32 board. Note: You have to hold down the "boot" button on the board (More info on "boot" button here).
  4. Hit the RST button.

Good luck! Hopefully this helped you out! 🍻
